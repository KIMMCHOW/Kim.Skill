# Private Overlay Example

This file is safe to commit. Copy its structure into `private-overlays/*.local.md` for local-only evidence and sharper self-review notes.

Files under `private-overlays/` are ignored by Git.

## Overlay Rules

- Keep raw chat excerpts, names, IDs, screenshots, and sensitive context here only.
- Do not paste this file into public `SKILL.md` or `references/research/`.
- When something should graduate into the public Skill, rewrite it as an abstract pattern, aggregate count, or anonymized rule.
- Add a date and source window to every private note so future KIM can tell whether the pattern is current or stale.
